# 📊 Netflix Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing Netflix data using Excel, SQL, and Python to extract meaningful insights such as content distribution, ratings, countries with most content, longest movies, and more.

The goal of this project is to demonstrate data analysis, data cleaning, and data visualization skills using different tools and technologies.

---

## 🛠️ Tools & Technologies Used
- Excel – Data Cleaning, Pivot Tables, Dashboard
- SQL (MySQL) – Data Querying and Analysis
- Python – Data Analysis using Pandas, Matplotlib, Seaborn
- Jupyter Notebook

---

## 📂 Dataset
- Dataset: Netflix Movies and TV Shows Dataset
- File: [netflix_titles.csv]{https://www.kaggle.com/datasets/shivamb/netflix-shows?resource=download}

---

## 📊 Analysis Performed (SQL)
Some of the SQL analysis performed in this project include:

1. Count the number of Movies vs TV Shows
2. Find the most common rating for Movies and TV Shows
3. Movies released in a specific year
4. Top 5 countries with the most content
5. Identify the longest movie
6. Content added in the last 5 years
7. Content by specific director
8. TV Shows with more than 5 seasons
9. Content without a director
10. Movies that are Documentaries

---

---

## 📊 Key Insights
- Netflix has more Movies than TV Shows.
- Most content is from the United States and India.
- The most common rating on Netflix is TV-MA.
- Documentaries are a major content category on Netflix.
- Content production increased significantly after 2015.
